# Lazio Open Data (dati-lazio-it)

Lazio Open Data is a regional government open-data portal for Italy running **CKAN** (~406 datasets).

**URL:** [https://raw.githubusercontent.com/api-evangelist/dati-lazio-it/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/dati-lazio-it/refs/heads/main/apis.yml)
